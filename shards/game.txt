But instead of “deal 2 damage”…

They say things like:

“Gravity Inversion: All units switch attack and health this turn.”

“Temporal Echo: Repeat the last unit played.”

“Solar Flare: All low-health units burn.”

“Dimensional Slip: Remove a unit from play for one turn.”

They feel environmental.
Not magical.

🎮 Why This Is Better Than Spells

Spells = “I cast fireball.”

Fractures = “The battlefield tears open.”

Way cooler.
Way more on-theme.
Way more brandable.

🌀 Types of Fractures

Keep it simple.

You want 4 categories:

1️⃣ Impact Fractures (Instant Effects)

Simple one-time changes.

Examples:

Shard Shock → Deal 2 to a unit.

Energy Surge → Gain 1 energy next turn.

Collapse → Destroy lowest HP unit.

These replace basic spell functions.

2️⃣ Environmental Fractures (1–2 turns)

Change battlefield rules briefly.

Examples:

Zero Gravity → Units can’t block.

Echo Field → Units entering play trigger twice.

Dark Eclipse → Healing is reversed.

Now strategy deepens.

3️⃣ Rift Fractures (Risk / Reward)

Unstable effects.

Examples:

Corruption Pulse → Buff random unit (could be enemy).

Unstable Mirror → Copy a unit randomly.

Wild Flux → Reduce all card costs by 1 this turn.

Streamer chaos fuel.

4️⃣ Persistent Fractures (Mini-Relics)

Last for 3 turns.

Examples:

Radiation Field → At end of turn, deal 1 to all units.

Accelerated Decay → Units lose 1 HP each turn.

Energy Feedback → Unused energy deals damage.

These create mid-game pivots.

🎯 What This Changes About Gameplay

Instead of:

“I use spell to remove your creature.”

It becomes:

“I destabilize the Nexus.”

It feels bigger.
Less generic.
More universe-defining.

⚖️ Keep It Simple for Beginners

Starter deck Fractures should be:

2–3 simple Impact Fractures

1 Environmental Fracture

No chaos ones yet

Advanced players unlock:

Rift Fractures

High variance meta-benders

🚀 Why This Is More Addictive

Because Fractures:

Create unpredictable moments

Encourage highlight clips

Reinforce your lore

Work beautifully with Ships & Harnesses

Example:

Now you're leaning into chaos.

Much stronger branding.

🚀 Core Ship Part Categories

Each Ship should have 6–8 hard slots.

You don’t want 20 types.
You want recognizable roles.

Here’s the clean structure:

1️⃣ Reactor Core (Power System)

Theme: Energy generation, overload, instability
Gameplay Focus: Resource manipulation

Examples:

Overcharge Core → If you spend all Energy, gain +1 next turn

Void Reactor → Fractures cost 1 less (once per turn)

Flux Matrix → 10% chance to gain bonus energy

These change tempo.

2️⃣ Engine Assembly (Mobility System)

Theme: Speed, repositioning, evasion
Gameplay Focus: Turn order, activation bonuses, tempo

Examples:

Rift Thrusters → First unit each turn gains “Strike”

Slipstream Drive → Once per match, replay a unit

Phase Engine → Units can’t be blocked first turn

These feel aggressive.

3️⃣ Weapon System (Offense System)

Theme: Offensive amplification
Gameplay Focus: Damage scaling

Examples:

Lance Array → Void units deal +1 damage

Fragment Cannon → When a unit dies, deal 1 to enemy Nexus

Corruption Emitter → First attack each turn corrupts target

These are straightforward but impactful.

4️⃣ Shield Matrix (Defense System)

Theme: Protection, mitigation
Gameplay Focus: Survival, stall

Examples:

Reactive Plating → First damage each turn reduced by 1

Null Barrier → Prevent first Fracture effect each match

Adaptive Hull → Gain shield when below 10 HP

These support control archetypes.

5️⃣ Command Uplink (Tactical System)

Theme: Strategic coordination
Gameplay Focus: Card draw, targeting, manipulation

Examples:

Tactical Array → Draw 1 when a unit evolves

Targeting Relay → Your Fractures choose targets

Combat AI → Once per turn, reroll a random effect

These reduce randomness.

6️⃣ Shard Interface (Nexus Interaction System)

Theme: Multiverse instability
Gameplay Focus: Fracture amplification

Examples:

Rift Amplifier → Environmental Fractures last +1 turn

Instability Conduit → 20% stronger corruption

Nexus Prism → Copy the first Fracture played each match

This ties deeply to your core theme.

7️⃣ Cargo / Utility Module (Wildcard Slot)

Theme: Salvage, flexibility
Gameplay Focus: Unique seasonal mechanics

Examples:

Salvage Drone Bay → When a unit dies, 20% chance to return to hand

Smuggler Cache → Start game with 1 extra card

Temporal Vault → Store 1 unused energy

This slot is where seasonal creativity happens.

8️⃣ Structural Frame (Passive Identity Slot)

Optional but powerful.

Theme: Ship architecture
Gameplay Focus: Global modifier

Examples:

Lightweight Frame → -1 max HP, +1 max Energy

Siege Frame → Units gain +1 health

Corrupted Frame → Units gain mutation chance

This slot defines personality.

🧠 Recommended Slot Count

For launch simplicity:

Use 6 slots:

Reactor

Engine

Weapon

Shield

Command

Shard Interface

Add Cargo later as expansion.
Add Structural Frame in Year 2.

Keep early system clean.

🎯 Why Categories Matter

Without categories:

Parts blur together

Players get confused

Balance explodes

With categories:

Players think:

“I need a better Reactor.”

That’s understandable.

⚖️ Balancing Philosophy

Each slot should:

Have 15–25 parts

Be swappable outside match

Not stack identical effects excessively

Avoid exponential scaling

No part should:
Double damage.
Break economy.
Auto-win.

They should:
Lean a deck.
Not replace it.
